# carvalhorr.github.io

Personal blog, served by GitHub Pages + Jekyll (minima theme).

- Posts go in `_posts/` as `YYYY-MM-DD-slug.md` with front matter
  (`layout: post`, `title:`). Drafts live in the separate private
  `blog` repo until they're published here.
- GitHub builds the site on push — no local tooling needed. To preview
  locally (optional): `gem install bundler jekyll && jekyll serve`.
